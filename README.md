# SoftWorks Website (Astro)

Basic website structure for SoftWorks, a fictional company that provides RAG solutions to accounting firms in Perth, Western Australia.

## Project Focus

- Company positioning and value proposition
- Core service overview
- Delivery process summary
- Contact call-to-action

## Commands

- `npm install`: Install dependencies
- `npm run dev`: Start development server on `localhost:4321`
- `npm run build`: Build static site output into `dist`
- `npm run preview`: Preview the production build locally

## Current Structure

- `src/layouts/Layout.astro`: Shared page shell and metadata
- `src/pages/index.astro`: SoftWorks homepage content sections
- `src/styles/global.css`: Global styles for layout and components
